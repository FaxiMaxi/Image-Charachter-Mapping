Simple tool to visualise and lable portions of image.
As example a picture of Marvel Charactes as a photo of assambled puzzles was chosen.

Tool to mark and lable regions: labelme https://github.com/wkentaro/labelme.git

html file is not working if opened locally in browser - use "python -m http.server 8000" in directory to start server an access the website at localhost:8000

Clicked area is highlighted and the label is shown at the bottom of the screem, clicking the label will lead to DuckDuckGo Image search of the highlighted charachter.
